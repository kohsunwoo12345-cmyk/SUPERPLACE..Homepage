# 👨‍🏫 선생님 등록 가이드

## 🔗 선생님 등록 페이지

**메인 페이지**: https://superplace-academy.pages.dev/students

---

## 📝 등록 방법

### 1단계: 원장 계정으로 로그인
```
URL: https://superplace-academy.pages.dev/login
이메일: kumetang@gmail.com
비밀번호: 1234
```

### 2단계: 학생 관리 페이지 접속
```
https://superplace-academy.pages.dev/students
```

### 3단계: 선생님 추가 버튼 클릭
페이지 하단의 **"선생님 관리"** 섹션에서:
- 🟣 **"선생님 추가"** 버튼 클릭

### 4단계: 선생님 정보 입력
팝업 모달에서 다음 정보를 입력:
- **이름**: 선생님 이름 (예: 홍길동)
- **이메일**: 로그인용 이메일 (예: hong@teacher.com)
- **비밀번호**: 초기 비밀번호 (선생님이 나중에 변경 가능)
- **전화번호**: 연락처 (선택사항)

### 5단계: 저장
- 🟢 **"선생님 추가하기"** 버튼 클릭
- 성공 메시지 확인: "✅ [이름] 선생님이 추가되었습니다!"

---

## 🎯 등록 후 권한 설정

### 1. 선생님 목록에서 해당 선생님 찾기
- 방금 추가한 선생님이 목록에 표시됩니다

### 2. 권한 설정 버튼 클릭
- 각 선생님 카드의 🔧 **"권한 설정"** 버튼 클릭

### 3. 권한 선택
두 가지 옵션 중 선택:

#### 옵션 A: 모든 학생 조회 가능
- 라디오 버튼: **"모든 학생 조회 가능"** 선택
- 결과: 원장과 동일하게 모든 학생 조회 가능

#### 옵션 B: 배정된 반만 공개 (권장)
- 라디오 버튼: **"배정된 반만 공개"** 선택
- 반 체크박스: 담당할 반 선택 (여러 개 가능)
- 결과: 선택한 반의 학생만 조회 가능

### 4. 저장
- 💾 **"저장"** 버튼 클릭
- 성공 메시지 확인

## 👥 선생님 로그인

선생님이 직접 로그인:

로그인 후:
- **학생 목록**: https://superplace-academy.pages.dev/students/list
- **배정된 반의 학생만** 표시됩니다 ✅

## 📊 API 엔드포인트 (참고용)

### 선생님 추가
```
POST /api/teachers/add
Content-Type: application/json

{
  "directorId": 1,
  "name": "홍길동",
  "email": "hong@teacher.com",
  "password": "password123",
  "phone": "010-1234-5678"
}
```

### 선생님 목록 조회
```
GET /api/teachers/list?directorId=1
```

### 권한 설정
```
POST /api/teachers/:id/permissions
Content-Type: application/json

{
  "directorId": 1,
  "permissions": {
    "canViewAllStudents": false,
    "canWriteDailyReports": true,
    "assignedClasses": [1, 2, 3]
  }
}
```

## 🚨 문제 해결

### "선생님 추가" 버튼이 안 보여요
- 원장 계정으로 로그인했는지 확인
- 페이지를 아래로 스크롤 (하단에 있음)

### 이메일이 중복된다고 나와요
- 해당 이메일이 이미 등록되어 있습니다
- 다른 이메일을 사용하세요

### 선생님이 로그인할 수 없어요
- 이메일/비밀번호 확인
- 원장이 제대로 등록했는지 확인
- 브라우저 캐시 삭제 후 재시도
