## 5️⃣ Add variable 팝업 (첫 번째 변수)

### 화면 설명
- 팝업 또는 모달 창이 열림
- 입력 필드와 체크박스

### 입력할 내용

```
┌──────────────────────────────────────────────────────┐
│  Add environment variable                            │
│                                                      │
│  Variable name                                       │
│  ┌──────────────────────────────────────────────┐   │
│  │ ALIGO_API_KEY                                │   │
│  └──────────────────────────────────────────────┘   │
│                                                      │
│  Value                                               │
│  ┌──────────────────────────────────────────────┐   │
│  │ 4bbi3l27pb5qh11tkujl578bttz6vb5j             │   │
│  └──────────────────────────────────────────────┘   │
│                                                      │
│  Environment                                         │
│  ☑️ Production   ☐ Preview                          │
│                                                      │
│  ☑️ Encrypt                                          │
│                                                      │
│  [Cancel]  [Add variable]                            │
│                     ↑                                │
│                  여기 클릭!                           │
└──────────────────────────────────────────────────────┘
```

**입력 순서:**
1. Variable name: `ALIGO_API_KEY` 입력
2. Value: `4bbi3l27pb5qh11tkujl578bttz6vb5j` 입력
3. Environment: `Production` 체크
4. Encrypt: 체크 ✅
5. `[Add variable]` 버튼 클릭

---

## 6️⃣ Add variable 팝업 (두 번째 변수)

### 화면 설명
- 첫 번째 변수를 추가하면 다시 목록으로 돌아옴
- 다시 `[Add variable]` 버튼 클릭

### 입력할 내용

```
┌──────────────────────────────────────────────────────┐
│  Add environment variable                            │
│                                                      │
│  Variable name                                       │
│  ┌──────────────────────────────────────────────┐   │
│  │ ALIGO_USER_ID                                │   │
│  └──────────────────────────────────────────────┘   │
│                                                      │
│  Value                                               │
│  ┌──────────────────────────────────────────────┐   │
│  │ wangholy                                     │   │
│  └──────────────────────────────────────────────┘   │
│                                                      │
│  Environment                                         │
│  ☑️ Production   ☐ Preview                          │
│                                                      │
│  ☐ Encrypt (체크 안 함)                              │
│                                                      │
│  [Cancel]  [Add variable]                            │
│                     ↑                                │
│                  여기 클릭!                           │
└──────────────────────────────────────────────────────┘
```

**입력 순서:**
1. Variable name: `ALIGO_USER_ID` 입력
2. Value: `wangholy` 입력
3. Environment: `Production` 체크
4. Encrypt: 체크 안 함 ☐
5. `[Add variable]` 버튼 클릭

---
