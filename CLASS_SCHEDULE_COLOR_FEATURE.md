# 반 관리 - 요일별 시간 및 색상 기능

## 완료된 기능

### 1. 요일별 시간 설정
- **위치**: https://superplace-academy.pages.dev/students/classes
- **기능**: 각 반에 대해 요일별로 다른 수업 시간 설정 가능
- **구현 내용**:
  - 반 추가/수정 모달에 7개 요일(월~일) 체크박스 추가
  - 각 요일별로 시작 시간과 종료 시간 입력 필드 제공
  - 체크된 요일에만 시간 입력 가능 (비활성화 상태 관리)
  - 데이터는 JSON 형식으로 `day_schedule` 컬럼에 저장
  
### 2. 반 색상 옵션
- **색상 팔레트**: 25가지 다양한 색상 제공
  - 보라(기본): #8B5CF6, #A78BFA, #C4B5FD
  - 파랑: #3B82F6, #60A5FA, #93C5FD
  - 청록: #06B6D4, #22D3EE, #67E8F9
  - 초록: #10B981, #34D399, #6EE7B7
  - 주황: #F97316, #FB923C, #FDBA74
  - 빨강: #EF4444, #F87171, #FCA5A5
  - 분홍: #EC4899, #F472B6, #F9A8D4
  - 노랑: #EAB308, #FACC15, #FDE047
  - 그레이: #6B7280
- **UI 표시**:
  - 반 카드 왼쪽 테두리에 색상 표시
  - 반 이름 옆에 색상 원형 아이콘 표시
  - 모달에서 색상 선택 버튼 (클릭하여 선택)

### 3. DB 스키마 변경
```sql
ALTER TABLE classes ADD COLUMN color TEXT DEFAULT '#8B5CF6';
ALTER TABLE classes ADD COLUMN day_schedule TEXT;
```

## 데이터 구조

### day_schedule JSON 형식
```json
{
  "월": { "start": "09:00", "end": "12:00" },
  "화": { "start": "14:00", "end": "17:00" },
  "수": { "start": "09:00", "end": "12:00" }
}
```

## UI 개선사항

### 반 목록 화면
- 각 반 카드에 선택된 색상 표시
- 요일별 시간 정보 표시 (예: "월: 09:00~12:00, 화: 14:00~17:00")
- 반 카드 좌측 테두리에 색상 적용

- 요일 체크박스: 월~일 7개
- 각 요일별 시간 입력: 시작 시간, 종료 시간
- 요일 체크 시 해당 시간 입력 활성화

## 사용 방법

### 새 반 추가 시
1. "새 반 추가" 버튼 클릭
2. 반 이름, 학년, 설명 입력
3. 색상 선택 (25가지 색상 중 선택)
4. 수업 요일 체크 (월~일 중 선택)
5. 각 요일의 시작/종료 시간 입력
6. "추가하기" 버튼 클릭

### 반 수정 시
1. 반 카드의 "수정" 버튼 클릭
2. 색상 변경 가능
3. 요일별 시간 변경 가능
4. "수정하기" 버튼 클릭
