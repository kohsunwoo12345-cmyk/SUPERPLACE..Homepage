# 우리는 슈퍼플레이스다 - 학원 마케팅 전문 웹사이트

## 프로젝트 개요
- **회사명**: 우리는 슈퍼플레이스다
- **목표**: 전국 학원장들을 위한 마케팅 교육 플랫폼 제공
- **주요 기능**: 
  - 네이버 플레이스 상위노출 교육
  - 블로그 마케팅 교육
  - 퍼널 마케팅 교육
  - 학원장 커뮤니티 (오픈채팅 & 오프라인 모임)

## 현재 완료된 기능
✅ **메인 페이지 (홈)**
- 히어로 섹션 (메인 비주얼)
- 통계 섹션 (500+ 학원, 95% 만족도 등)
- 핵심 교육 프로그램 소개 (3가지)
  - 네이버 플레이스 상위노출
  - 블로그 상위노출
  - 퍼널 마케팅
- Why Us 섹션 (차별점 강조)
- CTA (Call-to-Action) 섹션
- 반응형 네비게이션 (모바일 메뉴 포함)
- 푸터

## 현재 기능 진입 경로 (URI)

### 메인 페이지
- **경로**: `/`
- **설명**: 랜딩 페이지, 회사 소개 및 서비스 개요

### 네비게이션 링크 (준비 중)
- `/programs` - 교육 프로그램 상세 (미구현)
- `/success` - 성공 사례 (미구현)
- `/contact` - 문의하기 (미구현)
- `/login` - 로그인 (미구현)

## 추천 다음 개발 단계

### 1단계: 핵심 기능 구현 (우선순위 높음)
1. **문의 페이지** - 잠재 고객이 연락할 수 있는 창구 필요
2. **로그인/회원가입** - 회원 시스템 기반 구축
3. **교육 프로그램 상세 페이지** - 각 프로그램의 커리큘럼 상세 설명

### 2단계: 회원 전용 기능 (우선순위 중간)
4. **학원장 대시보드** - 로그인 후 이용 가능한 회원 전용 공간
5. **성공 사례 페이지** - 신뢰도 강화를 위한 사례 연구

### 3단계: 고급 기능 (우선순위 낮음)
6. **커뮤니티 기능** - 학원장들 간 소통 공간
7. **예약/결제 시스템** - 교육 신청 및 결제
8. **관리자 페이지** - 회원 관리, 문의 관리

## 로컬 개발 가이드

### 서버 시작
```bash
# 빌드
npm run build

# PM2로 서버 시작
pm2 start ecosystem.config.cjs

# 서버 상태 확인
pm2 list

# 로그 확인
pm2 logs webapp --nostream
```

### 개발 명령어
```bash
# 포트 정리
npm run clean-port

# 서버 테스트
npm run test

# 빌드
npm run build
```

## 사용자 가이드

### 방문자 (일반 사용자)
1. 메인 페이지에서 회사 소개 및 서비스 확인
2. "무료 상담 신청" 버튼 클릭 → 문의 페이지로 이동 (예정)
3. "교육 프로그램 보기" 버튼 클릭 → 프로그램 상세 페이지로 이동 (예정)

### 학원장 (회원)
1. "로그인" 버튼 클릭 → 로그인 페이지 (예정)
2. 로그인 후 대시보드 접근 (예정)
3. 교육 자료, 커뮤니티 등 이용 (예정)
